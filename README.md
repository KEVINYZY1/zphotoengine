# zphotoengine
zphotoengine是一个基本的PC版图像处理dll库，里面包含了动态库和静态库供大家调用。
调用库包含了PS所有图层混合处理算法，以及常用图像处理函数库。了解更多信息可以访问网站：www.xiusdk.com。
